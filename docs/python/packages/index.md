# Python Packages

## PyPI

PyPI (The Python Package Index) is a repository of software for the Python programming language that helps you find and install packages developed and shared by the Python community. You can also learn how to package your Python code for PyPI and browse projects and releases.

- [The Python Package Index](https://pypi.org/)
    - [Индекс пакетов Python](https://pypi.org/)

## NumPy
The fundamental package for scientific computing with Python

- [The NumPy official page](http://numpy.org)
- [The NumPy documentation](https://numpy.org/doc/)
- [The NumPy on GitHub](https://github.com/numpy/numpy/)
- [The NumPy on PyPI](https://pypi.org/project/numpy/)


## Pandas
Pandas is a tool for data analysis and manipulation, built on top of Python. Learn how to install, use and contribute to pandas, and explore its features, documentation and ecosystem.

- [The Pandas official page](https://pandas.pydata.org/)
- [The Pandas documentation](https://pandas.pydata.org/docs/)
- [The Pandas on GitHub](https://github.com/pandas-dev/pandas)
- [The Pandas on PyPI](https://pypi.org/project/pandas/)


## Scikit Learn
Scikit Learn is a Python module for machine learning built on top of SciPy and is distributed under the 3-Clause BSD license.

- [The Scikit Learn official page](https://scikit-learn.org/)
- [The Scikit Learn documentation](https://scikit-learn.org/stable/)
- [The Scikit Learn on GitHub](https://github.com/scikit-learn/scikit-learn)
- [The Scikit Learn on PyPI](https://pypi.org/project/scikit-learn/)
